# Sheeptag walking

Sheeptag walking is when undead or human style builders quickly build to block paths, it's unreliable but there are tricks you should know. You can only build about in melee range, therefore building 2x2 or 3x3 buildings won't make you backtrack to build. 4x4 is good to build if it would totally block a path or block vision.

Furthermore if you build 2 buildings the order of which building goes first is good to know as it will push your builder in a good position and the other way around will make you walk all the way around. It's good to remember this for draenei to base quicker.

Directions:

- If you are running east you must build south wall before the north wall. 
- If you are running north you must build east wall before the west wall. 
- If you are running south you must build west wall before the east wall. 
- If you are running west you must build north wall before the south wall.
